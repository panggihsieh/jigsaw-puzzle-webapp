# 🧩 96片拼圖遊戲 / Jigsaw Puzzle Web App

一款純網頁版的 96 片拼圖遊戲，支援自訂上傳圖片，無需安裝任何軟體，直接在瀏覽器中遊玩。

---

## ✨ 功能特色

- 🖼️ **自訂圖片**：可上傳任意圖片作為拼圖素材。
- 🔢 **96 片拼圖**：自動根據圖片比例切割為 8×12 或 12×8 的格局。
- ⏱️ **計時器**：遊戲開始後自動計時，完成時顯示花費時間。
- 🎨 **深色玻璃質感 UI**：美觀現代的介面設計。
- 🖱️ **拖放操作**：使用滑鼠拖曳碎片至正確位置完成拼圖。

---

## 🚀 使用方式

1. 下載或 Clone 本專案：
   ```bash
   git clone https://github.com/panggihsieh/jigsaw-puzzle-webapp.git
   ```

2. 直接用瀏覽器開啟 `index.html`（不需要任何伺服器）。

3. 點擊「**上傳圖片**」按鈕，選擇一張圖片。

4. 圖片載入後，點擊「**開始遊戲**」，碎片會隨機打散到右側。

5. 將右側碎片**拖曳**到左側拼板的正確位置，全部完成後即獲勝！
